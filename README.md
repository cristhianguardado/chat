# Chat

Chat based in a MEAN app

### Tech

* [Mongodb]
* [Express] 
* [Angular]
* [Node.js]
* [Socket.io]

### Installation

```sh
$ npm install
$ cp .env.example .env
$ npm start (for development)
```
### Docker

```sh
$ cp .env.example .env
$ docker-compose up -d
```

### Swagger
```sh
$ npm install
$ cp .env.example .env
$ npm start (for development)
```
